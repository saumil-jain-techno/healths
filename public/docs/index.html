Introduction

Start (and never finish) side projects with this API.

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Login

Example request:

    "http://127.0.0.1:8000/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Api-Version: v2" \
    --data "{
    \"email\": \"test@example.com\",
    \"password\": \"Saumil#Jain@123\"
}"

    "http://127.0.0.1:8000/api/login"

User management

APIs for managing users

Store a user.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/user/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --header "Api-Version: v2" \
    --form "name=Saumil"\
    --form "email=test@example.com"\
    --form "password=Saumil#Jain@123"\
    --form "image=@/tmp/phpo1CMIQ" 
const url = new URL(
    "http://127.0.0.1:8000/api/user/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
    "Api-Version": "v2",
};

const body = new FormData();
body.append('name', 'Saumil');
body.append('email', 'test@example.com');
body.append('password', 'Saumil#Jain@123');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
